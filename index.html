Sobre a BZuk

Conheça a linguagem que está revolucionando a programação no Brasil

Linguagem de Programação Brasileira

BZuk é uma linguagem de programação criada por um desenvolvedor brasileiro com ascendência japonesa, combinando a eficiência do C++ com a simplicidade do Python e Lua.

Com sintaxe em português e palavras abreviadas de forma intuitiva, BZuk é perfeita para iniciantes e experts que desejam uma linguagem rápida e expressiva.

Rápida como C++ e fácil como Python
Sintaxe em português com palavras abreviadas
Perfeita para iniciantes em programação
Poucas linhas para fazer muito (igual Lua)

Exemplos de Código

Veja como é fácil e intuitivo programar com BZuk

exemplo.bz BZuk 1.7 
// BZuk - Interpretador Beta Alfa 1.8

let nome = ler="Qual seu nome?"
const idade = ler="Qual sua idade?", numero
let status = "ativo"
const pais = "Brasil"

impri("nome, idade, status, pais")

nome = "Outro nome"      // Ok, let pode mudar
// idade = 99            // Vai dar erro, pois é const
status = "inativo"       // Ok

rp(cidade)
impri("cidade")

se idade MN 18,
    impri("Você é maior de idade!")
senão,
    impri("Você é menor de 18!")

repi-3,
    impri("Loop rápido!")

Características Principais

Descubra o que faz da BZuk uma linguagem especial

Sintaxe em Português

Use palavras em português e abreviações intuitivas como "impri" para print e "ler" para input.

Alta Performance

Velocidade comparável ao C++ com uma sintaxe muito mais simples e amigável.

Fácil de Aprender

Perfeita para iniciantes, com conceitos simplificados e documentação em português.

Poucas Linhas

Escreva programas complexos com menos código, seguindo a filosofia do Lua.

Tipagem Flexível

Sistema de tipos que oferece flexibilidade quando precisa e segurança quando necessário.

Feita no Brasil

Orgulho brasileiro com influência da precisão e eficiência japonesa.
