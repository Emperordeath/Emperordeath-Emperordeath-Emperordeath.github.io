Sobre o BZuk

BZuk é uma linguagem criada para tornar o aprendizado de programação acessível, com sintaxe em português e comandos intuitivos. O projeto é pequeno e ágil: feito por uma única pessoa (15 anos) e em rápido desenvolvimento — feedback e colaboradores são bem-vindos no Discord.

Versões planejadas

Versão 1 — "Python-like" (ATUAL)

Interpretador linha-a-linha, sintaxe simples e legível. Ideal para aprendizado e prototipagem. Mais lenta mas muito fácil de usar.

  • • Lançamento inicial (em desenvolvimento)
  • • Código simples, foco em clareza

Versão 2 — "C++-like" (FUTURA)

Mesma sintaxe visual, mas com backend compilado/otimizado para performance. Será desenvolvida após finalizar a primeira versão.

  • • Melhor performance e consumo de memória
  • • Ideal para produção e scripts complexos

let nome = ler="Qual seu nome?" const idade = ler="Qual sua idade?", numero let status = "ativo" const pais = "Brasil" impri("nome, idade, status, pais") rp(cidade) impri("cidade") se idade MN 18, impri("Você é maior de idade!") senão, impri("Você é menor de 18!") repi-3, impri("Loop rápido!")
