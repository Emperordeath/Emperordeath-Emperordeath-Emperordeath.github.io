Sobre a BZuk

Conheça a linguagem que está sendo desenvolvida com orgulho brasileiro e precisão japonesa

1 ano de desenvolvimento - Fase Beta

BZuk é uma linguagem de programação que está em desenvolvimento há 1 ano, atualmente em sua fase beta inicial. Esta primeira versão é baseada em Python, com planos para uma versão nativa em C++ no futuro.

Com sintaxe em português e palavras abreviadas de forma intuitiva, BZuk é perfeita para iniciantes e experts que desejam uma linguagem rápida e expressiva.

Primeira versão em Python - Futura versão em C++ com alta performance
Sintaxe em português com palavras abreviadas intuitivas
Poucas linhas para fazer muito (igual Lua)
Orgulho brasileiro com influência da precisão japonesa

Exemplos de Código

Veja como é fácil e intuitivo programar com BZuk - Linguagem em fase beta com recursos limitados

exemplo.bz BZuk 1.7 Beta 
// BZuk - Interpretador Beta 1.8
// Linguagem em desenvolvimento - Versão atual em Python

let nome = ler="Qual seu nome?"
const idade = ler="Qual sua idade?", numero
let status = "ativo"
const pais = "Brasil"

impri("nome, idade, status, pais")

nome = "Outro nome"      // Ok, let pode mudar
// idade = 99            // Erro! Constante não pode ser alterada
status = "inativo"       // Ok

rp(cidade)               // Ler input para variável cidade
impri("cidade")          // Imprimir valor da variável cidade

se idade MN 18,          // Se idade Maior ou igual (MN) a 18
    impri("Você é maior de idade!")
senão,
    impri("Você é menor de 18!")

repi-3,                  // Repetir 3 vezes
    impri("Loop rápido!")

Características Principais

Descubra o que faz da BZuk uma linguagem especial - Mesmo em fase inicial de desenvolvimento

Sintaxe em Português

Use palavras em português e abreviações intuitivas como "impri" para print, "ler" para input e "se" para if.

Alta Performance Futura

Versão atual em Python. Próxima versão em C++ para velocidade comparável com sintaxe simplificada.

Fácil de Aprender

Perfeita para iniciantes, com conceitos simplificados e documentação em português.

Poucas Linhas

Escreva programas complexos com menos código, seguindo a filosofia do Lua.

Tipagem Flexível

Sistema de tipos que oferece flexibilidade quando precisa e segurança quando necessário.
